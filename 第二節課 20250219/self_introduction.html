Self Introduction

我的個人網站：https://www.jw-albert.tw/

在學期間，我積極參與各類專案，並嘗試透過微創業來建立一些網站與 AI 應用，希望能將技術轉化為實際價值。此外，我也具備管理伺服器的經驗，熟悉伺服器維護與最佳化，並曾自行開發 Python 應用（https://www.fcuems.tw/），我的 GitHub 主頁為 JW-Albert，以解決特定需求。

對於未來，我希望能在加拿大攻讀碩士學位，並進一步拓展國際視野與專業技能。此外，我計劃在學術研究的同時，累積產業經驗，參與開源專案，並建立自己的技術品牌。我也對創業充滿興趣，希望未來能夠結合人工智慧、資訊安全與雲端技術，開發具創新價值的產品，並在國際市場上發展自己的事業。

除了學業與技術專長外，我也是一名業餘無線電愛好者，具備專業無線電通訊設備操作經驗，熟悉數位與模擬通訊技術。此外，我擁有初級救護員（EMT-1）資格，並參與衛生福利部醫療通訊（ARESV）計畫，以支援緊急應變與通訊協調工作。我相信技術的發展能夠帶來更多可能性，因此我始終保持學習與探索的精神。

如果有機會，我希望能與各位交流學習，拓展更廣的視野，也期待未來能與志同道合的夥伴共同合作，開創更多可能性！
